## Install vagrant in windows:
After download and install vagrant, run these commands below:

>PS D:\> vagrant box add centos/7
>
>PS D:\> vagrant box add ubuntu/xenial64
>
>PS D:\> vagrant box update
>PS D:\> vagrant box list
